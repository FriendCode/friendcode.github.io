FriendCode was created in 2013 with the vision of creating powerful development tools, tackling issues like collaboration, cloud development and many others.

We are building Codebox, an open source Cloud IDE, than can run only on the cloud, but on your desktop or your server as well. Codebox.io lets you host and manage Codebox instances online as a service using a dashboard or an API.
